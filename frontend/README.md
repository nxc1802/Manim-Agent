# Manim Agent Frontend

React, TypeScript, and Vite dashboard for projects, scene editing, pipeline activity,
artifact history, and render jobs.

```bash
npm install
npm run dev
```

The frontend expects `VITE_API_BASE_URL`, `VITE_SUPABASE_URL`, and
`VITE_SUPABASE_ANON_KEY` in the appropriate `.env` file.

## Quality checks

```bash
npm run lint
npm test
npm run build
```
